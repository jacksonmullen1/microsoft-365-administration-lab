# Ticket 002: Group Management

## Issue

The organization required department-based groups to simplify access management and collaboration.

## Action Taken

Created Security Groups for IT, HR, Sales, Finance, and Operations. Created a Microsoft 365 Group for All-Employees and assigned users to the appropriate groups based on department.

## Result

Department-based access groups were successfully configured to support identity management, collaboration, and future permission assignments.

## Skills Demonstrated

* Microsoft Entra ID (Azure AD)
* Security Groups
* Microsoft 365 Groups
* Group membership management
* Identity and Access Management (IAM)
