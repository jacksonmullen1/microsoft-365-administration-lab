# Ticket 010: Intune Configuration Profile

## Issue

The organization required standardized Windows security settings across managed devices.

## Action Taken

Created a Windows configuration profile enforcing password policies, BitLocker encryption, and additional security settings. Assigned the profile to the Corporate Devices security group.

## Result

Windows devices can receive consistent configuration settings that align with organizational security standards.
