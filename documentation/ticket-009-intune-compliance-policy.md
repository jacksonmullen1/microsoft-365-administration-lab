# Ticket 009: Intune Compliance Policy

## Issue

The organization required Windows devices to meet minimum security requirements before accessing corporate resources.

## Action Taken

Created a Windows compliance policy enforcing BitLocker, Secure Boot, TPM, Microsoft Defender Antivirus, and password requirements. Assigned the policy to the Corporate Devices security group.

## Result

Devices can be evaluated against organizational security standards before being granted access to Microsoft 365 resources.

* Microsoft Intune
* Compliance Policies
* Endpoint Security
* Windows Security
