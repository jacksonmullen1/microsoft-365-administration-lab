# Lessons Learned

## Overview

This project provided hands-on experience administering Microsoft 365 Business Premium in a simulated enterprise environment. The lab focused on common administrative tasks performed by Help Desk Technicians, Desktop Support Specialists, and Microsoft 365 Administrators.

Throughout the project, I configured cloud identity services, messaging, collaboration platforms, security controls, and endpoint management while following real-world IT administration workflows.

# Key Takeaways

This project strengthened my understanding of Microsoft 365 administration by demonstrating how Microsoft Entra ID, Exchange Online, SharePoint Online, OneDrive, Intune, and Conditional Access work together to support identity management, collaboration, endpoint security, and day-to-day IT operations.

The experience also reinforced the importance of following structured administrative processes, applying security best practices, and documenting changes in a way that supports enterprise IT environments.
