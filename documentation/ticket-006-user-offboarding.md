# Ticket 006: User Offboarding

## Issue

An employee departed the organization and required immediate removal of Microsoft 365 access.

## Action Taken

Blocked user sign-in, reset the account password, removed Microsoft 365 licensing, and removed the user from organizational groups.

## Result

The user's access to Microsoft 365 resources was successfully revoked and the license was reclaimed for future use.

## Skills Demonstrated

* User lifecycle management
* Microsoft Entra ID
* License management
* Identity and Access Management
* Security administration
