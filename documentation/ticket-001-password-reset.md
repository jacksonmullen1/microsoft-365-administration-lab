# Ticket 001: Password Reset

## Issue

Maria Lopez was unable to sign in and required a password reset.

## Action Taken

The user's password was reset through the Microsoft 365 Admin Center. The option to require the user to change their password at the next sign-in was enabled.

## Result

The user's account access was restored successfully.

* Microsoft 365 Admin Center
* User account management
* Password reset process
* Help desk troubleshooting
