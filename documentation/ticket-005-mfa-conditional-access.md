# Ticket 005: Multi-Factor Authentication & Conditional Access

## Issue

The organization required stronger identity security to protect user accounts from unauthorized access.

## Action Taken

Enabled Microsoft Security Defaults and configured a Conditional Access policy requiring Multi-Factor Authentication (MFA) for employee accounts accessing Microsoft 365 cloud resources.

## Result

User accounts now require additional authentication during sign-in, significantly improving account security.

## Skills Demonstrated

* Microsoft Entra ID
* Conditional Access
* Multi-Factor Authentication (MFA)
* Identity Security
* Zero Trust Security
