# Ticket 003: Shared Mailbox Configuration

## Issue

The IT department required a centralized mailbox for handling support requests.

## Action Taken

Created a shared mailbox named **IT Support** and delegated access to the IT Support Specialist.

## Result

The shared mailbox allows multiple authorized users to access and manage incoming support requests without requiring a separate user license.

* Microsoft Exchange Online
* Shared mailbox administration
* Mailbox delegation
* Microsoft 365 Admin Center
